# 🖥️ Projeto MontePC - Simulador de Hardware

Sistema web inteligente desenvolvido para auxiliar na montagem de computadores de alta performance, validando compatibilidade de peças e calculando o consumo energético (TDP) em tempo real. 

Projeto desenvolvido como parte do curso de Engenharia da Computação do Instituto Federal de Mato Grosso (IFMT).

## 🚀 Funcionalidades
* **Catálogo Dinâmico:** Mais de 100 peças reais (AM5, DDR5, etc.) cadastradas.
* **Validador de Engenharia:** Bloqueia combinações incompatíveis (ex: Soquetes diferentes, Fontes mais fracas que o consumo total).
* **Autocompletar Inteligente:** Seleção de Kits pré-montados com possibilidade de personalização peça por peça.
* **Sistema de Usuários (CRUD):** Criação de conta para salvar e gerenciar builds personalizadas.
* **Exportação Profissional:** Geração de orçamento oficial em formato PDF.

## 🛠️ Tecnologias Utilizadas
* **Backend:** Python e Django (Framework MTV)
* **Frontend:** HTML5, CSS3 e Bootstrap 5
* **Banco de Dados:** SQLite
* **Bibliotecas Extras:** `xhtml2pdf` (Geração de documentos)
